⚡ FastShare

The Future of Offline File Sharing

A sophisticated, cross-platform solution for high-speed local data transfer.

Project Overview

FastShare is built to bridge the gap between platforms when it comes to sharing large files without relying on slow cloud services or expensive data.

  • Built with Flutter
  • Production-Ready
  • Zero-latency architecture

Core Concept

Local Network First.

By leveraging LAN and WiFi technologies, FastShare creates a direct tunnel between devices, ensuring maximum privacy and speed.

Key Feature: Offline Sharing

No internet? No problem.

FastShare works entirely offline, making it perfect for remote areas, planes, or high-security environments.

Key Feature: QR Pairing

Connecting should be effortless.

Just scan a QR code to securely pair devices. No more searching for IP addresses or pairing codes.

True Cross-Platform

Seamless sharing across all your devices.

  • Android (Material 3)
  • Windows
  • macOS & Linux

Security: Password Protection

Protect your shared files with industrial-grade hashing.

Users can set optional passwords for shares, preventing unauthorized access even on open networks.

Security: Secure Tokens

Token-based authentication ensures that once access is granted, the session remains secure and isolated.

Performance: 50GB+ Support

Optimized for the "Big Data" era.

Stream-based file handling allows for files larger than 50GB without crashing or freezing the UI.

Material 3 UI/UX

A modern, clean aesthetic.

Following the latest design standards with Teal accents and dynamic theme support (Dark/Light mode).

The Technical Stack

  • Flutter: Single codebase, multiple platforms.
  • Riverpod: Modern state management.
  • Local HTTP Server: High-performance file serving.

Persistence with Hive

Blazing fast NoSQL database.

Used for transfer history, ensuring your records survive app restarts with zero overhead.

Production Quality

Rigorous fixes implemented:

  • Deduplication of history
  • Permission compliance (Android 14)
  • Splash screen optimization
  • Memory-safe streaming

System Architecture

Sender: Hosts a temporary HTTP Server.

Receiver: Connects via browser or app client.

Protocol: Secure local streaming.

Future Roadmap

  • WiFi Direct / Hotspot Auto-pairing
  • Encrypted P2P Channels
  • Cloud Backup Integration (Optional)
  • Community Plugins Support

Conclusion

FastShare is ready to revolutionize how you move data.
